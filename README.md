# Passion

Passion 是一个 Windows 桌面个人助理应用，基于 Tauri + React 构建。当前定位是轻量工作台，把提醒、翻译、网络检测、SSH 隧道、下载、本机自动化脚本和常用实用工具集中到一个桌面应用里。

## 已实现功能

### 工作台

- 首页以功能卡片展示当前能力。
- 顶部导航可快速切换模块。
- 支持最小化到系统托盘。
- 支持设置是否开机启动。

### 提醒

- 新增、查看、启用/停用、删除提醒。
- 到点后显示桌面通知，并弹出应用提醒窗口。
- 支持单次提醒。
- 支持每天重复提醒。
- 支持中国法定工作日提醒。
  - 内置 2026 年中国法定节假日和调休日历。
  - 运行时不依赖网络。
  - 如果选择的日期不是法定工作日，会自动顺延到下一个法定工作日同一时间。
  - 触发后会自动排到下一个法定工作日。

### 翻译

- 支持 OpenAI Chat Completions 兼容接口。
- 可配置 Base URL、模型名和 API Key。
- 支持测试 AI 连接状态。
- 适配本地部署模型，例如 Ollama、LM Studio 或其他兼容 `/v1/chat/completions` 的服务。

### 网络检测

- TCP 端口检测，判断指定 host:port 是否开放。
- 支持对指定主机进行 TCP 端口范围扫描，显示进度和开放端口。
- 本机端口占用查看，显示占用进程和 PID。
- 显示检测结果和响应耗时。

### SSH 隧道

- 支持本地端口转发（`ssh -L`）。
- 使用系统 OpenSSH，不内置 SSH 协议实现。
- 支持配置 SSH 程序路径，未配置时自动检测环境变量 PATH 中的 `ssh.exe` / `ssh` 并保存。
- 支持私钥文件认证。
- 支持启动、停止、编辑和删除隧道。
- 列表状态精简为运行中和已停止，异常退出的隧道按已停止展示并可重新启动。
- 隧道列表底部固定显示总数、运行中和已停止统计。
- 使用数据库保存隧道配置，运行状态由应用后台进程管理。
- 启动前检查本地端口占用和私钥文件是否存在。
- 关键操作和异常会写入 `passion.log`，便于定位 SSH 路径、端口占用、私钥、认证和进程退出问题。

### 下载工具

- 支持 HTTP/HTTPS 地址、本地文件路径和局域网共享文件。
- 支持选择保存目录、暂停、继续和取消任务。
- 支持下载进度、速度、耗时、保存路径、文件大小和任务状态显示。
- 支持基于 `.part` 临时文件的续传。

### 脚本任务

- 新增、删除、启用/停用脚本任务。
- 支持立即运行一次。
- 支持按固定分钟间隔定期执行。
- 支持每天固定时间执行。
- 支持每周指定星期和时间执行。
- 支持在执行命令中填写脚本路径、可执行文件路径和参数。
- 支持 Windows 脚本和可执行文件：
  - `.ps1`
  - `.bat`
  - `.cmd`
  - `.exe`
- 保存最近一次执行状态：
  - 开始时间
  - 结束时间
  - 退出码
  - stdout 摘要
  - stderr 摘要
  - 错误信息
- 列表筛选和统计聚焦全部、运行中、等待执行、已停用；上次执行失败的已启用任务归入等待执行并保留错误信息。
- 同一个任务运行中时，不会并发启动第二次。

### 实用工具

- 支持 Base64 编码和解码。
- 支持 Hex 编码和解码。
- 支持秒级 / 毫秒级时间戳转换。
- 支持复制转换结果。
- 工具内容区直接展示输入、结果和操作控件，减少重复说明。

## 法定工作日日历

当前内置日历文件：

```text
src-tauri/resources/calendars/cn-workdays-2026.json
```

数据来源为中国政府网国务院办公厅发布的 2026 年节假日安排通知。应用运行时直接读取打包进程序的本地数据，不需要联网。

后续年份需要在国务院发布安排后更新对应 JSON，再重新打包发布。

## 脚本执行边界

脚本任务功能会执行本机文件，因此当前实现做了以下限制：

- 不执行 shell 拼接后的任意命令字符串；UI 会把输入拆分为程序路径和参数，再由后端按文件类型执行。
- 只允许 `.ps1`、`.bat`、`.cmd`、`.exe`。
- stdout/stderr 只保存摘要，避免日志无限增长。
- 应用未运行时不会执行脚本任务。

## 开发

安装依赖：

```bash
npm install
```

启动前端开发服务器：

```bash
npm run dev
```

启动 Tauri 开发模式：

```bash
npm run tauri dev
```

运行前端测试：

```bash
npm test -- --run
```

运行 Rust 测试：

```bash
cd src-tauri
cargo test
```

## 构建

构建前端：

```bash
npm run build
```

构建 Windows 安装包：

```bash
npm run tauri build
```

构建完成后，安装包通常位于：

```text
src-tauri/target/release/bundle/msi/
src-tauri/target/release/bundle/nsis/
```

## 当前限制

- 法定工作日日历目前只内置 2026 年。
- 下载工具暂不支持多线程下载。
- 脚本任务只在应用运行时执行，不是系统后台服务。
- 翻译功能依赖用户自行配置可用的 OpenAI 兼容服务。
