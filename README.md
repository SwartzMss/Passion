# Passion

Passion 是一个 Windows 桌面个人助理应用，基于 Tauri + React 构建。当前定位是轻量工作台，把提醒、翻译、网络检测、下载、系统监控和本机自动化脚本集中到一个桌面应用里。

## 已实现功能

### 工作台

- 首页以功能卡片展示当前能力。
- 顶部导航可快速切换模块。
- 支持最小化到系统托盘。
- 支持设置是否开机启动。

### 提醒

- 新增、查看、启用/停用、删除提醒。
- 到点后显示桌面通知，并弹出应用提醒窗口。
- 支持单次提醒。
- 支持中国法定工作日提醒。
  - 内置 2026 年中国法定节假日和调休日历。
  - 运行时不依赖网络。
  - 如果选择的日期不是法定工作日，会自动顺延到下一个法定工作日同一时间。
  - 触发后会自动排到下一个法定工作日。

### 翻译

- 支持 OpenAI Chat Completions 兼容接口。
- 可配置 Base URL、模型名、API Key 和默认目标语言。
- 适配本地部署模型，例如 Ollama、LM Studio 或其他兼容 `/v1/chat/completions` 的服务。

### 网络检测

- TCP 端口检测，判断指定 host:port 是否开放。
- 本机端口占用查看，显示占用进程和 PID。
- 显示检测结果和响应耗时。

### 下载工具

- 支持 HTTP/HTTPS 地址、本地文件路径和局域网共享文件。
- 支持选择保存目录、暂停/继续和取消任务。
- 显示保存路径、文件大小、速度和任务状态。

### 系统监控

- 查看 CPU 使用率。
- 查看内存已用/总量。
- 查看磁盘已用/总量。
- 查看系统运行时长。
- 支持手动刷新。

### 脚本任务

- 新增、删除、启用/停用脚本任务。
- 支持立即运行一次。
- 支持按固定分钟间隔定期执行。
- 支持 Windows 脚本和可执行文件：
  - `.ps1`
  - `.bat`
  - `.cmd`
  - `.exe`
- 保存最近一次执行状态：
  - 开始时间
  - 结束时间
  - 退出码
  - stdout 摘要
  - stderr 摘要
  - 错误信息
- 同一个任务运行中时，不会并发启动第二次。

## 法定工作日日历

当前内置日历文件：

```text
src-tauri/resources/calendars/cn-workdays-2026.json
```

数据来源为中国政府网国务院办公厅发布的 2026 年节假日安排通知。应用运行时直接读取打包进程序的本地数据，不需要联网。

后续年份需要在国务院发布安排后更新对应 JSON，再重新打包发布。

## 脚本执行边界

脚本任务功能会执行本机文件，因此当前实现做了以下限制：

- 不支持在 UI 里输入任意命令字符串。
- 不支持脚本参数。
- 只允许 `.ps1`、`.bat`、`.cmd`、`.exe`。
- stdout/stderr 只保存摘要，避免日志无限增长。
- 应用未运行时不会执行脚本任务。

## 开发

安装依赖：

```bash
npm install
```

启动前端开发服务器：

```bash
npm run dev
```

启动 Tauri 开发模式：

```bash
npm run tauri dev
```

运行前端测试：

```bash
npm test -- --run
```

运行 Rust 测试：

```bash
cd src-tauri
cargo test
```

## 构建

构建前端：

```bash
npm run build
```

构建 Windows 安装包：

```bash
npm run tauri build
```

构建完成后，安装包通常位于：

```text
src-tauri/target/release/bundle/msi/
src-tauri/target/release/bundle/nsis/
```

## 当前限制

- 法定工作日日历目前只内置 2026 年。
- 下载工具第一版暂不支持进度条、暂停、续传和多线程下载。
- 脚本任务只在应用运行时执行，不是系统后台服务。
- 翻译功能依赖用户自行配置可用的 OpenAI 兼容服务。
